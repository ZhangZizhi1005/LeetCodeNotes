### 暴力解：N指针 ###

对于N个数之和等于target之类的题目，我们始终可以用N个指针分别指向不同的N个元素。遍历数组求解。

- 时间复杂度随N的增大迅速增大：O(n<sup>N</sup> )
- 同时这个方法无法避免重复，当题目要求返回的解不能重复的时候，还需要对解进行去重，还需要额外的花费。
